How does it work?
  1. You enter your register number and password to etlab. The app will store these data locally on your phone.
  2. For getting attendance data, this register number and password is send over a secure connection to our mediating server. Mediating Server is managed by us and it never logs username nor password.
  3. The mediating server will send the username and password to etlab and get necessary pages. The server will process these pages, extract necessary data, compresses it and then send it over to your app over secure connection.
  4. The app will render the attendance using recieved data.
  5. Steps 2-4 are repeated whenever you reopen the app. The heavy lifting(downloading full webpages, scraping and processing) is done by the mediating server, thus making the app lighter on memory and network.
